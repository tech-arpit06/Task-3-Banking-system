# Banking System

## Project Description
This project is a simple Banking System developed using Object-Oriented Programming concepts. It allows customers to create accounts, deposit money, withdraw money, transfer funds, and view transaction history.

## Features
- Create Customer Account
- Deposit Money
- Withdraw Money
- Fund Transfer
- Check Account Balance
- View Transaction History

## Classes Used

### Customer
Stores customer details such as:
- Customer ID
- Customer Name

### Account
Stores account information:
- Account Number
- Balance

### Transaction
Stores transaction details:
- Transaction Type
- Amount
- Date and Time

## Working

1. User creates an account.
2. User can deposit money.
3. User can withdraw money.
4. User can transfer money to another account.
5. System stores transaction history.
6. User can check account balance anytime.

## Example

Initial Balance = ₹5000

Deposit ₹2000
Balance = ₹7000

Withdraw ₹1000
Balance = ₹6000

Transfer ₹1500
Balance = ₹4500

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Functions
- Lists
- Conditional Statements
- User Input Handling

## Outcome
The project successfully simulates basic banking operations and maintains customer account details and transaction records.
